Available API Endpoints

  • GET/POST /restaurant/menu/ - List and create menu items
  • GET/PUT/DELETE /restaurant/menu/{id}/ - Retrieve, update, delete menu item
  • GET/POST /api/tables/ - List and create bookings
  • GET/PUT/DELETE /api/tables/{id}/ - Retrieve, update, delete booking
  • POST /auth/users/ - User registration
  • POST /auth/token/login/ - User login (get token)
  • POST /auth/token/logout/ - User logout
