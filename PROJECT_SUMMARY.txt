===========================================
NEXT STEPS TO DEPLOY
===========================================

1. INSTALL DEPENDENCIES
   Open Command Prompt in project folder and run:
   > pip install -r requirements.txt

2. SETUP MYSQL DATABASE
   - Create a database named 'littlelemon' in MySQL
   - Update database credentials in littlelemon/settings.py

3. RUN MIGRATIONS
   > python manage.py makemigrations
   > python manage.py migrate

4. CREATE ADMIN USER
   > python manage.py createsuperuser

5. TEST THE APPLICATION
   > python manage.py runserver
   Visit: http://127.0.0.1:8000/

6. RUN TESTS
   > python manage.py test

7. PUSH TO GITHUB
   Option A: Double-click push_to_github.bat
   Option B: Manual commands (see DEPLOYMENT_GUIDE.txt)

===========================================
TESTING WITH INSOMNIA/POSTMAN
===========================================

Step 1: Register User
POST http://127.0.0.1:8000/auth/users/
Body: {"username": "testuser", "password": "testpass123"}

Step 2: Login
POST http://127.0.0.1:8000/auth/token/login/
Body: {"username": "testuser", "password": "testpass123"}
Save the token from response

Step 3: Test Menu API
GET http://127.0.0.1:8000/restaurant/menu/
Header: Authorization: Token {your_token}

Step 4: Create Menu Item
POST http://127.0.0.1:8000/restaurant/menu/
Header: Authorization: Token {your_token}
Body: {"title": "Pizza", "price": "12.99", "inventory": 10}

Step 5: Test Booking API
GET http://127.0.0.1:8000/api/tables/
Header: Authorization: Token {your_token}

Step 6: Create Booking
POST http://127.0.0.1:8000/api/tables/
Header: Authorization: Token {your_token}
Body: {"name": "John", "no_of_guests": 4, "booking_date": "2024-12-25T19:00:00Z"}
